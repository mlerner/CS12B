Purpose: The purpose of this program is to implement a grocery list program that stores data about the different grocery item objects.
Function: This program can construct a new grocery list and add up to 10 items to the list. It can then find the total cost of all the items in the list by adding the price per unit multiplied by quantity. The GroceryItemOrder class constructs order objects with the attributes of name, quantity, and price per unit. You can change the quantity per unit within this class.
